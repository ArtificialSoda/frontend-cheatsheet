creating html file

At the root of every webpage on the internet is an HTML file. At their core, HTML files hold a fundamental structure that must be included when making your own webpages.

The first line of an HTML document should always be <!DOCTYPE html>. The doctype declaration is an instruction to the browser about what type of document to expect and which version of HTML is being used. As of now, it’s HTML5.

The <html> element, the root of an HTML document, should be added after the !DOCTYPE declaration. All content/structure for an HTML document should be contained between the opening and closing tags.

The <head> element contains general information about an HTML page that isn’t displayed on the page itself. This information is called metadata and includes things like the <title> of the HTML document and links to stylesheets.

The <body> element represents the content of an HTML document. Content inside those tags is rendered on the web browsers, i.e. is visible.

                        
<!DOCTYPE html>
<html lang="en">
<head>
    <title>Example</title>
</head>
<body>
    <p>Hello World!</p>
</body>
</html>
